# 2026-Manajemen Ruang Backend

## Deskripsi
Project ini merupakan backend untuk Sistem Peminjaman Ruangan Kampus yang dikembangkan menggunakan ASP.NET Core.  
Backend ini bertugas untuk mengelola data ruangan dan proses peminjaman ruangan.

## Fitur
- CRUD data Ruangan
- Koneksi database MySQL
- Menggunakan Entity Framework Core
- REST API

## Teknologi yang Digunakan
- ASP.NET Core
- Entity Framework Core
- MySQL
- Git & GitHub

## Status
Saat ini sistem masih dalam tahap pengembangan.
